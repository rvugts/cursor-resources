# Cursor Vibe Coding Setup

A comprehensive collection of Cursor rules, slash commands, MCP server configs, templates, and workflows for efficient development.

## Overview

This repository contains:

- **Cursor Commands**: Slash commands for common development tasks and meta prompting
- **Cursor Rules**: Language-specific and workflow-specific rules
- **Templates**: Project templates and documentation templates
- **MCP Servers**: Configuration examples for Model Context Protocol servers
- **Prompts**: Reusable prompts for various development tasks
- **Workflows**: Step-by-step workflow guides
- **Cheatsheets**: Quick reference guides

## Quick Start

1. Copy `.cursorrules` from `cursor-settings/` to your project root
2. Copy relevant rule files from `cursor-rules/` to your project root in .cursor/rules
